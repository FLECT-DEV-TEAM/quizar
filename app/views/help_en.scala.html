Usage for quiz master.

At first, the quiz master makes a room. The room is the place to publish quiz.

You can name the room as you want.(e.g. "Java study", "Hollywood movie quiz")

Next, you should make some questions.

The question can take 5 choices. And it can have an explanation for the answer and reference link.

After that, let's hava an event with some quiz panelists.

In the event, you can publish some quiz to the panelists. Th event makes a ranking of panelists.

Though the quiz master and the panelists are not necessary to be in the same place, maybe it is more fun to in same place.

Usage for quiz panelists.

At first, the quiz panelist enter the room of quiz master.

In the room, you can chat with others.

When the quiz master starts an event, the entry dialog is shown. After entry the event, you can answer the quiz.

The time limit for a quiz is 10 seconds.

This app requires good network environment.

At the time of quiz, if your device is sleeping, or your browser is closed, you can not answer the quiz. It's a part of the game.

If posting question is allowed by room setting, anyone can make a question and post it to room owner.

Edit event.

You can show this page by "Edit event" menu.

If the event is not started, you can modify event settings and start event.

If the event is running, event entry are listed. You can finish event.

Capacity
Capacity of the event. Currently the max number of capacity is limited to 50.
Title
Title of the event.
Start date
Specify the start date of the event.
Start time
Specify the start time of the event.
Passcode

If specified, passcode is required to entry event.

Description
Any description.
  • There is no necessary field in event settings. So you can start event without input anything.
  • If you specify the start date and time, the event doesn't start automatically at that time. These fields are used to make recent event list.
  • If the running event spend 24 hours without publishing question, it automatically close.
  • You can not modify past event information.

Entry the event

When the room owner starts the event, others in the room see the confirm dialog of event entry.

You can entry the event by it.

  • You also entry the event from user menu.
  • Event entry is first come, first served.If the number of event entry is reached to the capacity, following entry is rejected.
  • You can not cancel the entried event.
  • If you don't entry the event, you can see the question, but can not answer.

Answer the question

When the room owner publish the question, it shows in others device.The time for choosing answer is 10 seconds.

The number of answerer is shown in the button.

Ranking

You can see this page by "Ranking" menu. Or from shortcut bar of the bottom.

Now!

The ranking of currently running event.

If the event is not running, it isn't shown.

Event
The winners list of past event. You can look the detail ranking by clicking the row.
Total
Total ranking of the room. You can look the detail of the user by clicking the row.

The event ranking is calced by following step.

  1. The number of correct answer.
  2. If it is same, the time of correct answer.

When the event finish, top 10 user get the points.

The total ranking is calced by following step.

  1. Point
  2. If it is same, the number of correct answer.
  3. If those are same, the time of correct answer.
